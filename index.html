PIP

Presentations, landings, portfolios, and longforms.

  • PIP para el control de modulos en python

    En primer lugar comporbaremos si tenemos pip instalado en nuestro equipo, ya que es la herramienta que nos permitirá instalar el modulo que usaremos en el taller. Para ello usaremos:

      pip --version

  • Version de PIP

    Para ver que versión de pip tenemos instalada actualmente. 

      sudo apt-get install python-pip    
      sudo apt-get install python3-pip

Comenzamos por la instalación

En primer lugar instalaremos el módulo pyTelegramBotAPI para poder interactuar con la API de Telegram desde Python.

En algunos casos (habitualmente windows si no está pip añadido en el path) necesitaremos instalar el modulo de esa otra manera.

  pip install pyTelegramBotAPI    
  python -m pip install pyTelegramBotAPI
